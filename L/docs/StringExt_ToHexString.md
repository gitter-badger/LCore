# ToHexString

##### Summary

            Takes an array of Bytes and returns a friendly hexadecimal string.
            Ex. Byte[] { 10, 50, 80, 120, 150, 200, 250, 250 }  ->  "0x0A32507896C8FAFA"
            This method cannot fail.
